jQuery Twister Demo

Example #1

Usage case with default options.

$('#twister-example_1 .image').twister();

Example #2

Usage case with overflown rotation.

$('#twister-example_2 .image').twister({
  overflown: true
});

Example #3

Usage case with full rotation.

$('#twister-example_3 .image').twister({
  degrees: 360,
  overflown: false,
  scale: 1.5
});

Example #4

Usage case with full, overflown rotation.

$('#twister-example_4 .image').twister({
  degrees: 360,
  overflown: true,
  scale: 1
});
